This folder contains some scripts to estimate expost Inequality of Opportunity,
as well as some tools for graphical analysis.

It also contains the scripts to run a Shapley value decomposition.

data: csv with mocking data.

R files: 

expost_iop.R : R script where the main code is run.
shapley.R : R script to run the Shapley value decomposition
functions_exante : functions needed to run ex ante IOp
functions_expost : functions needed to run ex post IOp
functions_shapley : functions needed to run the shapley value decomposition

Pdf files:
Functions_expost.pdf : Support documentation for ex post IOp estimation
Functions_shapley.pdf : Support documentation for Shapley decomposition
